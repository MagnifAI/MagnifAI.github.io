// I don't just use AI — I build the harnesses that make it ship real software

Most people prompt AI. I orchestrate agents — a fleet of specialized AI workers that decompose work, execute in parallel, verify each other's output, and ship production code without a human in the loop.

My workflow: an orchestrator decomposes the task, then specialist agents — planning, backend, database, QA, review — execute in parallel waves with verification gates (types, tests, adversarial review) between every stage. Knowledge bases persist across projects, so every build makes the next one faster.

The payoff: one engineer operating with the throughput of a team — without giving up code review, tests, or operational discipline.
